## Project Overview
This project presents a robotic vehicle designed for emergency response scenarios such as police and fire rescue missions. The vehicle is remotely controlled using a Raspberry Pi touchscreen-based device and streams live video using a mounted camera.

The goal is to reduce manpower risk in hazardous environments.

---

## Key Features
- Forward, Backward, Left, Right movement
- 200 meter operational range
- Live camera streaming
- Touchscreen GUI using Pygame
- Wi-Fi communication between Raspberry Pi units
- 3-hour battery operation target

---
